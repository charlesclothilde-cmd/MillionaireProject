Millionaire Project

A public Streamlit app for exploring EuroMillions tickets through model scoring, behavioral crowding risk, and honest historical backtesting.

Open the app View on GitHub
2004-2026 EuroMillions draw history
5 tabs Generate, analyze, model, backtest, research
0 claims No promise of prediction or profit

Built for analysis, not superstition.

The app combines ticket generation with behavioral signals and historical replay, so each strategy can be inspected before it is trusted.

Generate

Use the best current ranked variant, anti-crowd generation, or simpler historical strategies.

Analyze Ticket

Score birthday bias, lucky-number pull, round-number pull, playslip patterns, and crowding risk.

Backtesting

Replay strategies against historical draws with prize tiers, value estimates, and match distributions.

Important disclaimer

EuroMillions draws are designed to be random. This project is an educational and exploratory analysis tool, not financial advice, gambling advice, or a reliable way to predict future results. Treat generated tickets as entertainment.
